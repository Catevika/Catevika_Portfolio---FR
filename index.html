Support Desk

Voir le site Support Desk Voir la démo Voir le code

Aperçu

Problème : Une grande entreprise accueille chaque nouvel employé avec 1 ordinateur portable, 1 écran de bureau, 1 clavier, 1 téléphone portable, 1 tablette, 1 badge pour la porte des employés, 1 badge pour la cafétéria, 1 ebook d'accueil. Les techniciens ont besoin d'une plate-forme pour gérer le support quotidien pour tous ces équipements.

Solution, un site avec 2 modules :

  • d'un côté, les employés peuvent envoyer des tickets, recevoir et répondre à des notes, gérer leur profil personnel
  • de l'autre, les administrateurs IT peuvent gérer les produits, recevoir des tickets, envoyer et recevoir des notes ainsi que gérer les profils des employés

Ce que j'ai appris

  • Remix (compilateur, gestionnaire HTTP côté serveur, serveur, navigateur) + React 18 + React Router 6 + Typescript
  • Prisma (ORM Node.js et TypeScript de nouvelle génération) + MongoDB
  • Déploiement sur Vercel

ShopLineArt

Voir le site ShopLineArt Voir la démo Voir le code Serveur Voir le code Client

Aperçu

Problème : Une blogueuse a besoin d'illustrer ses articles. Elle aime le site Pexels.com, site de stockage gratuit de photos d'artistes étonnants. Elle aimerait passer moins de temps à sélectionner et à télécharger ces photos.

Solution, un site avec 3 caractéristiques principales :

  • recherche de photos par mot-clé,
  • affichage des photos en différentes tailles,
  • téléchargement protégé de photos, dossier au format .zip avec crédits associés

Ce que j'ai appris

  • React + React-router-dom V6
  • React Context
  • Auth vs routes publiques + MongoDB / Mongoose
  • Client de l'API Pexels Photo
  • Express
  • Theme clair / sombre CSS
  • EmailJS pour le formulaire de contact
  • JSzip + file-saver pour ajouter des fichiers texte aux photos téléchargées
  • Lien Serveur - Client lors du déploiement sur Render.com
